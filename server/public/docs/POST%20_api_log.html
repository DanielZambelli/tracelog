    • POST /api/log

    Creates a log within the service instance.
    Requires authorised session @see POST /auth
    Accepts content-type: application/json.
    Returns response: 200 OK

    Body Parameters:
    Name Type Description
    type string error, message, warning
    data string the data that the log should include
    cat string category to make the log easily identifiable, for example "payment"
